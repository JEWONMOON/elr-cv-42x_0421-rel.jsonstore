## 📌 Usage Instructions

* 각 파일은 상호작용에 대한 구체적 기록을 포함합니다.
* `.md` 파일은 Markdown 형식으로 되어 있으며, 가독성이 좋습니다.
* `.json` 파일은 구조화된 데이터로, 프로그램적 접근이 용이합니다.
